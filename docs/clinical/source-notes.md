# Source Notes

## Implemented in v1

- `Kyriakopoulou 2017` is the primary exact-source implementation for:
  - brain biparietal diameter
  - brain fronto-occipital length
  - skull biparietal diameter
  - skull occipitofrontal diameter
  - transverse cerebellar diameter
  - atrial diameter
  - vermian height
  - vermian width
- `Harreld 2011` is the primary exact-source implementation for:
  - corpus callosum length

## Important implementation details

- `Kyriakopoulou 2017` values are implemented from the extracted supplement calculator formulas.
- `Harreld 2011` mean is the published quadratic fit.
- `Harreld 2011` sigma is derived from the paper's published 95% individual prediction intervals.

## Deferred for later extraction

- cavum septum pellucidum width
- pons AP diameter
- third ventricle width
- Chiari II posterior-fossa measurements
- qualitative findings such as absent CSP or non-visualized corpus callosum
- head-circumference-driven microcephaly / macrocephaly cards
